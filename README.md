# swift-cli-gradle-xcode-hello-world

## Description
A POC for gradle build for swift.
The Dockerfile uses multi stage build.
This will only compile not run.

## Tech stack
- gradle
- swift

## Docker stack
- gradle:jdk11
  - swift

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
